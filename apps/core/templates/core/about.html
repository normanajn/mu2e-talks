About

Mu2eTalks

Fermilab Scientific Computing Division

Mu2eTalks is a web application developed by staff at Fermi National Accelerator Laboratory (Fermilab) for use by the Scientific Computing Division. It provides a structured way for Mu2e personnel to log weekly effort talks against projects, categories, and performance goals, and allows administrators and group leaders to filter, review, export, and summarize that activity using both standard reports and AI-assisted narrative summaries.

Internal use only. This application is intended exclusively for Fermilab Mu2e collaboration members. Unauthorized access or use is prohibited.
